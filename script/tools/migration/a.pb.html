Chaque icône sur la carte représente un lieu, en cliquant dessus vous obtenez le détail dans une une petite fenêtre. Dans cette fenêtre, si vous cliquez sur le nom du lieu, une fiche descriptive apparaît en bas de votre écran. Si plusieurs lieux sont trop proches les uns des autres, une seule icône apparaîtra. Pour les distinguer, il suffit de "zoomer" sur le carte.

Vous pouvez déplacer la carte en cliquant sur celle-ci et en bougant la souris. Pour "zoomer", appuyez sur les bouton [+] et [-] sur la gauche.

J'attire grandement votre attention sur le fait que cette rubrique dynamique a aussi besoin de vous pour s'enrichir... Faites-nous parvenir vos fiches descriptives des lieux que vous avez visités, ainsi que les photos.
