Focus sur la Corse Romane

Construites sur un plan basilical, avec tres peu d'ouvertures, les eglises romanes de Corse surprennent d'abord le voyageur qui les juge austeres. Pourtant, bien vite, elles le seduisent, tant elles se marient harmonieusement avec leur ecrin naturel. Les reliefs enigmaniques qui les ornent ajoutent a leur charme un parfum de mystere.
